# Changelog

## 1.1.0

- Added subtle speed-responsive camera distance and optional dynamic FOV.
- Added aim-focused composition for bows, crossbows, tridents, spears, shields, and spyglasses.
- Added a corrected third-person crosshair that projects the unchanged vanilla eye-ray target.
- Moved shoulder positioning ahead of projection and frustum preparation to keep culling aligned with the rendered camera.
- Added Motion & HUD settings with live preview in English and Simplified Chinese.
- Added motion/projection tests, stable build-tool pinning, and continuous integration on Java 25.
